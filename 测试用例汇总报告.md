## 业务规则说明

### 准入判断规则
| 条件 | 要求 |
|------|------|
| 月工资 | monthly_salary > 10000 |
| 社保状态 | social_security_flag = 1 (有社保) |
| **结果** | **必须同时满足以上两条才准入** |

### 计算系数规则
| 场景 | 条件 | 计算系数 |
|------|------|----------|
| 准入境况 + 资产为正 | monthly_salary > 10000 AND social_security_flag = 1 AND avg_3m_balance > 0 | 2.3 |
| 准入境况 + 资产为负/零 | monthly_salary > 10000 AND social_security_flag = 1 AND avg_3m_balance <= 0 | 0.5 |
| 不准入境况 | 未满足准入条件 | 0 |

### 授信额度计算公式
```
credit_limit = user_level × avg_3m_balance × monthly_salary × 计算系数
若结果为负数，则返回 0.00
```
